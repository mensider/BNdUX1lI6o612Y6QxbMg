Plan Epsilon
------------
[.] Label every single location in the grid.  <---
[.] Calculate the shortest path between any two locations in the grid and store the path.
[.] Implement an anti-collision check before each poll.
[.] Retrieve the current bot location and destination from server.
[.] Search for the shortest path from storage.
[.] Send command next direction of motion.

[+] Easily Scalable in number of bots, for a constant grid.
[-] Initial path calculation can be exhaustive for a large grid.
[-] Difficult to scale the grid.
